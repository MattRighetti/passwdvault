## Getting started
### Installing
First of all you'll need to install [Go](https://golang.org)

#### Installing PasswdVault Command Line Tool Manually
```sh
$ git clone https://github.com/MattRighetti/passwdvault.git
$ cd passwdvault
$ make install
```
